# 🍀 오늘의 운세 - High School Edition

고등학생들을 타겟으로 한 가볍고 산뜻한 '오늘의 운세' 웹 서비스입니다. 
포인트 컬러 `#93C572`(피스타치오 그린)를 사용하여 편안하고 풋풋한 느낌을 줍니다.

## 📁 폴더 구조
team-project/
├── index.html        # 메인 웹 페이지 (상단바, 입력화면, 결과화면 포함)
├── css/
│   └── style.css     # 반응형 스타일 및 애니메이션 적용
├── js/
│   └── script.js     # 화면 전환 및 해시 기반 고정 운세 알고리즘
├── images/
│   ├── main-banner.png   # 메인 화면 일러스트
│   └── card-image-1.png  # 결과 화면 상단 일러스트
└── README.md         # 프로젝트 설명 파일

## ✨ 핵심 기능
1. **화면 전환 방식 (SPA 느낌):** 버튼을 누르면 페이지 이동 없이 메인 화면이 숨겨지고 결과 화면이 부드럽게 나타납니다.
2. **반응형 디자인:** 스마트폰, 태블릿, PC 어디서 접속해도 화면 크기에 맞게 예쁘게 배열됩니다.
3. **고정 결과 시스템 (Hash 기반):** '입력한 생년월일 + 오늘 날짜'를 조합하여 해시값을 만듭니다. 같은 사람이 오늘 여러 번 확인해도 **항상 똑같은 결과**가 나오며, 다음 날이 되면 새로운 운세로 갱신됩니다.
4. **맞춤형 콘텐츠:** 매점, 체육복, 립밤, 코인 노래방 등 10대들이 일상에서 공감할 수 있는 키워드들로 행운의 아이템/장소를 구성했습니다.

## 🚀 사용 방법
`index.html` 파일을 크롬 등 웹 브라우저에서 열어 바로 테스트해 볼 수 있습니다. `images` 폴더 안에 원하는 이미지를 교체해 넣으면 더욱 완성도 높은 프로젝트가 됩니다.
